# Multidistrict Analysis

Five-district (Ahmednagar, Solapur, Nashik, Bellary, Dharwad) production and weather analysis.

**Inputs:** `data/multidistrict_production_area_series.csv`, `data/multidistrict_weather_aggregated.csv`

**Output:** `code/multidistrict/data/multidistrict_panel.csv`, figures in `figures/`

**Run:**
```bash
python code/multidistrict/run_multidistrict_pipeline.py
```
